# unsuck-java

Complementary materials for the course **[Effective Java Software Design](https://devchampions.com/training/java)**. 

### Requirements
- [x] Docker
- [x] Java 15+

#### Running the app in dev mode
```
./gradlew generateJooq
./gradlew bootRun
```

#### Running tests
```
./gradlew test
```

#### Code formatting
```
./gradlew spotlessApply
```
